# Featured Projects

## Windows Helpdesk & Support Tools

### [ISG-Desk](https://github.com/Silviu3369/ISG-Desk)

A Windows desktop helpdesk tool for local network diagnostics and technician support.

The project focuses on practical IT support tasks such as ping checks, DNS checks, gateway and internet validation, Wi-Fi analysis, printer discovery, SNMP device inspection, and report export.

Main areas:

- Windows 10/11 local diagnostics
- Network triage for helpdesk technicians
- DNS, gateway, internet, and adapter checks
- Printer discovery and support
- Wi-Fi analysis
- HTML, TXT, and JSON report export
- Local-first design with no cloud dependency

---

### [ISG-Printer](https://github.com/Silviu3369/ISG-Printer)

A technician-focused Windows printer support tool.

The goal of this project is to simplify corporate printer support by helping technicians discover print servers, install printers, manage queues, inspect printer status, and read printer information through SNMP.

Main areas:

- Print server discovery
- Printer installation from server queues
- Local printer discovery
- Spooler and queue diagnostics
- SNMP toner, model, serial number, page count, and status reads
- Portable technician workflow
- Windows WPF application architecture

---

### [ISG-Doctor](https://github.com/Silviu3369/ISG-Doctor)

A Windows diagnostics and repair tool for workstation support.

This project focuses on giving technicians a clear overview of a Windows machine before applying repairs. It separates read-only diagnostics from repair actions and supports structured troubleshooting.

Main areas:

- Windows health scan
- Disk, performance, security, network, battery, and event log checks
- Health score and technician findings
- DISM, SFC, DNS flush, Winsock reset, cleanup, and Windows Update cache repair actions
- HTML report export
- Controlled repair workflow

## Cisco Network Infrastructure Projects

### [Cisco-Blaze-project](https://github.com/Silviu3369/Cisco-Blaze-project)

A Cisco Packet Tracer enterprise network lab built around two main office locations.

The project demonstrates core enterprise networking concepts such as segmentation, redundancy, routing, and security.

Main areas:

- Multi-site network design
- VLAN segmentation
- OSPF routing
- HSRP redundancy
- ASA firewall basics
- Network documentation and simulation

---

### [HQ-Branch-Secure-Network](https://github.com/Silviu3369/HQ-Branch-Secure-Network)

A Cisco Packet Tracer lab simulating a secure HQ and Branch enterprise environment.

The project includes routing, redundancy, firewalling, DMZ design, NAT, ACLs, DHCP, and site-to-site VPN connectivity.

Main areas:

- HQ and Branch topology
- Cisco ASA firewall configuration
- OSPF Area 0
- HSRP gateway redundancy
- DMZ with public-facing services
- NAT and ACL policies
- Site-to-Site IPSec VPN
- DHCP services
- SSH restricted to management VLANs
- STP security features

---

### [Corporate-Multi-Zone-Network](https://github.com/Silviu3369/Corporate-Multi-Zone-Network)

A larger enterprise-grade multi-zone network design using a secure and redundant dual-hub architecture.

The project connects multiple enterprise zones using DMVPN over IPsec, logical VLAN segmentation, HSRP, ASA firewalls, Zone-Based Firewall policies, and centralized management services.

Main areas:

- Four-zone enterprise network design
- Dual-hub DMVPN architecture
- IPsec encrypted connectivity
- OSPF routing
- HSRP redundancy
- ASA firewall and Zone-Based Firewall policies
- VLAN segmentation
- Centralized NTP, Syslog, SSH, and DHCP services
- Structured IP addressing plan
